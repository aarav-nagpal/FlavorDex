# FlavorDex 🍽️

FlavorDex is a modern Recipe Finder web application built using HTML, CSS, and JavaScript.

It allows users to search recipes, filter by categories, sort results, and save favorite dishes through an interactive and responsive user interface.

---

## 🚀 Features

- 🔍 Search recipes
- 🍗 Filter recipes by categories
- ❤️ Favorites section
- 📋 Recipe cards with details
- ↕️ Sorting functionality
- 📱 Responsive UI Design
- ⚡ Dynamic rendering using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
